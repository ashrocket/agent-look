How it works

Three steps. Seriously.

1

Set your screenshots folder

Run the installer. Point it at wherever macOS saves your screenshots. Done once.

2

Take screenshots normally

Nothing changes. ⌘⇧4, ⌘⇧5, Simulator — all work.

3

Say /look in Claude

Claude finds your recent screenshots, previews them, and only loads what you pick.

What it actually does

Smart, not heavy.

  • 🔍
    Bypasses iCloud TCC restrictions Uses mdfind kMDItemIsScreenCapture — Spotlight metadata works even on iCloud paths where find throws "Operation not permitted".
  • 🤏
    Token-efficient by design Shows you a list first. You pick which screenshots to load. Images are read in a isolated haiku subagent — never bloating your main context.
  • ✏️
    Auto-renames generic filenames "Screenshot 2026-03-22 at 2.15.32 PM.png" becomes "2026-03-22_1415_login-error-state.png" — based on what's actually in the image.
  • 🔌
    Works in Claude Code and Claude Desktop Installs as a Claude Code plugin or as an MCP server for Claude Desktop. One installer handles both.

Install

Claude Code (plugin)

# add the marketplace and install
/plugin marketplace add https://github.com/ashrocket/agent-look
/plugin install agent-look@agent-look-dev

Claude Desktop (MCP)

# clone and run the installer
git clone git@github.com:ashrocket/agent-look.git
cd agent-look && bash install.sh
