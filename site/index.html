How it works

Three steps. Seriously.

1

Install once

brew install agent-look then agent-look register. Wires up Claude, Gemini, and Codex automatically.

2

Take screenshots normally

Nothing changes. ⌘⇧4, ⌘⇧5, Simulator — all work.

3

Say /look

Finds screenshots from the last 3 minutes, reads text via Spotlight OCR, auto-renames generic filenames, and reports back. No prompts needed.

What it actually does

Smart, not heavy.

  • 🔍
    Bypasses iCloud TCC restrictions Uses mdfind -onlyin … kMDItemIsScreenCapture=1 — Spotlight metadata works on iCloud paths where find throws "Operation not permitted".
  • 🫥
    Handles the U+202F narrow no-break space macOS uses a visually-identical-but-different space before AM/PM in screenshot names. Shell mv silently fails on it. The MCP server uses fs.renameSync() with exact Spotlight byte paths — no shell, no problem.
  • 🤏
    Token-efficient by design Uses Spotlight OCR (kMDItemTextContent) to read screenshot text for free — no vision tokens. Falls back to a thumbnail examiner (resized to 800px) only when OCR is empty. Most screenshots never need vision at all.
  • ✏️
    Auto-renames without asking "Screenshot 2026-03-22 at 2.15.32 PM.png" becomes "2026-03-22_1415_login-error-state.png" — fully automatic, no prompts. Skips files that already have descriptive names.
  • 🔌
    Works everywhere One MCP server, four platforms. agent-look register wires up Claude Code, Claude Desktop, Gemini CLI, and Codex in one command.

Install

Homebrew (recommended)

# install and register in all detected AI tools
brew tap ashrocket/agent-look
brew install agent-look
agent-look register

Registers the MCP server in Claude Code, Claude Desktop, Gemini CLI, and Codex automatically.
agent-look status to check, agent-look unregister to remove.

Claude Code plugin

# install as a plugin (includes /look skill + screenshot-examiner agent)
/plugin marketplace add https://github.com/ashrocket/agent-look
/plugin install agent-look@agent-look-dev

Manual (git clone)

# clone and run the installer — sets up Claude Desktop MCP
git clone git@github.com:ashrocket/agent-look.git
cd agent-look && bash install.sh
