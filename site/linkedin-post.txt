Tired of dragging Screenshot-2026-03-22-at-2.15.32-PM.png into the Claude command line and then having to explain what was wrong?

Same. I am so lazy.

Well now you don't have to drag those files — or add them to Claude Desktop either.

Set your screenshot directory once. Take your screenshots. Come back to Claude and say /look. It finds what you captured in the last 3 minutes, reads them in a subagent (so your context stays clean), renames the generic filenames based on what's actually in the image, and asks if you want to use any of them for the current task.

Bonus: macOS screenshot filenames contain a "narrow no-break space" (U+202F) before AM/PM that looks exactly like a regular space but breaks every shell rename command. I found out the hard way. The MCP server handles it correctly. You're welcome.
